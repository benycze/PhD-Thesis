This repository contains LaTeX source codes of my PhD thesis. The toppic of my work belongs to 
the area of high-speed computer networks. My goal is to generate a processing pipeline from abstract description.


*** RELEASE NOTES OF LATEX TEMPLATE ***

2015
========
This is modification of the previous version (2012). Main differences:
* chapter and section headings, page headers and footers
* theorems, examples, proofs etc. based on amsthm package
* numbering based on section (may be changed)
* specified just after \begin{document}
* based on memoir document class
